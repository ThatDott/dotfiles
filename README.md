# Dev Setup Nako Bai

This repo will serve as my **tech dump**. Tanan nga akong nagamit ug pohon gamiton, diri ninyo makita!

## **The tools I currently use:**
- Tinapolan Nvim Setup
    - [Neovim](https://github.com/neovim/neovim)
    - [LazyVim](https://github.com/lazyvim/lazyvim)
    - [Kickstarter.nvim](https://github.com/nvim-lua/kickstart.nvim)

- Gusto'g Linux vibe sa Windows
    - [GlazeWM](https://github.com/glzr-io/glazewm)
    - [ZeBar](https://github.com/glzr-io/zebar)
    - [WSL2](https://github.com/microsoft/WSL)
    - [Rainmeter](https://github.com/rainmeter/rainmeter)

- Para chuyans imong command prompt
    - [oh-my-posh](https://github.com/jandedobbeleer/oh-my-posh)
